# Mail API
	API for simple form website, only send mail.

## Route
: /send
	@method POST	


	@params: 
	    message: string , 
	    name: string, 
	    company: string, 
	    phone: string, 
	    mail: string

	#### message: 
		This field is Text in body mail
	
	#### name: 
		Username of user sended mail
	
	#### company: 
		Name company to user sended mail
	
	#### phone: 
		Number phone of user sended mail
	
	#### mail: 
		Mail for contact user sended mail
